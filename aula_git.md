#Aula sobre Git

>Data: 29/11/2019

Iniciando com o Git.

Para instalar o GIT use:

''''bash
	$ apt-get install git 
''''
Para adicionar arquivos a seremm monitorados use:

''''bash
	$git add
''''
Para criar um registro do projeto use:

''''bash
$ git commit -m "comentario"
''''
